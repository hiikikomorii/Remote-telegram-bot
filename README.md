# Remote System Contol Bot
### English-version
### Commands
* **Show all commands ```/help```**
* **Execute system commands ```/run```**
* **Take screenshots ```/scr```**
* **Type text automatically ```/write```**
* **Press keyboard keys ```/key```**
* **Show Windows notifications ```/notify```**

### Usage
1. **Set your _TOKEN_ and _ADMIN_ID_ in the script**
2. **Run the bot: ```py RemoteBot.py```**
3. **Use commands in Telegram to control your system**

### Security
* **Only the _ADMIN_ID_ can execute commands**
* **Keep your token private**
